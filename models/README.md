# Каталог моделей

Проект не предусматривает подкачки моделей с использованием torch.hub, huggingface_hub и других хабов в связи с ограничениями трафика скачивания на территории РФ.
Рекомендуем воспользоваться любыми собственными доступными методами скачивания с локальным хранением моделей в данном каталоге.

Загрузите сюда необходимые для проекта модели:
